# 📘 Gemini Prompt Gallery 운영 가이드

이 문서는 사이트를 관리하는 방법과 자주 묻는 질문(FAQ)을 정리한 매뉴얼입니다.

---

## 1. 데이터 업데이트 방법 (Data Entry)
프롬프트나 내용을 추가하고 싶을 때 이 순서대로 진행합니다.

### 📝 텍스트 입력 (방법 1: 추천)
- **파일 위치**: `c:\project\geminipromptc\DATA_INPUT.md`
- 간단한 수정이나 소량의 데이터 입력 시 추천합니다.

### 📊 엑셀/CSV 입력 (방법 2: 대량 등록)
데이터가 많을 경우 엑셀을 사용하는 것이 훨씬 빠릅니다.

1.  **템플릿 파일 열기**: `c:\project\geminipromptc\prompts.csv` 파일을 엑셀이나 구글 스프레드시트로 엽니다.
2.  **데이터 작성**:
    - `title`: 제목
    - `category`: 부서명 (원하는 대로 작성 가능)
    - `image`: 이미지 파일명 (예: `marketing-01.png`)
    - `description`: 짧은 설명
    - `prompt`: 긴 프롬프트 내용
3.  **CSV 저장**:
    - 엑셀에서 **다른 이름으로 저장** -> 파일 형식을 **CSV (쉼표로 분리) (*.csv)** 로 선택해서 저장합니다.
    - 저장 위치는 원래 있던 곳(`c:\project\geminipromptc\prompts.csv`)에 덮어쓰세요.
4.  **알려주기**: "CSV 파일 업데이트했어"라고 말해주시면 제가 변환 작업을 수행합니다.

### 🖼️ 이미지 추가
- **파일 위치**: `c:\project\geminipromptc\public\images\`
- **방법**: 준비한 이미지 파일을 이 폴더에 붙여넣으세요.
- **파일명 규칙**: 영문 소문자와 숫자 사용을 권장합니다. (예: `sales-01.png`, `hr-template.jpg`)

### 🚀 배포 (사이트 반영)
1. 내용 작성이 끝나면 AI(저)에게 **"수정했어"** 또는 **"업데이트 해줘"**라고 말해주세요.
2. 제가 변경사항을 저장소에 올리면(Push), 약 1~2분 뒤에 사이트가 **자동으로 업데이트** 됩니다.

## 2. 자주 묻는 질문 (FAQ)

### Q. 데이터를 얼마나 많이 넣을 수 있나요?
**A. 걱정하지 않으셔도 됩니다.**
- 수백 개까지는 속도 저하 없이 거뜬합니다.
- 현재 계획하신 50~100개 규모에서는 아주 쾌적하게 작동합니다.

### Q. 카테고리(부서명)를 바꾸려면 코드를 수정해야 하나요?
**A. 아니요, 코딩은 필요 없습니다.**
- 데이터 입력 시 `부서` 항목에 원하시는 이름을 적기만 하면 됩니다.
- 예: `Marketing` → `Global Marketing`으로 적으면 메뉴도 자동으로 바뀝니다.

### Q. 이미지는 주소로 넣나요, 파일로 넣나요?
**A. 파일 파일로 넣는 것이 가장 확실합니다.**
- 인터넷 주소 링크는 원본이 사라지면 엑박(깨진 이미지)이 뜰 수 있습니다.
- `public/images` 폴더에 직접 파일을 넣어두면 사이트와 함께 영구적으로 보존됩니다.

### Q. 실수로 데이터를 잘못 입력하면 사이트가 고장나나요?
**A. 대부분 안전합니다.**
- 다만, 따옴표(`"`)를 닫지 않거나 하는 문법 오류가 있으면 화면이 안 뜰 수 있습니다.
- 문제가 생기면 저에게 "동작하지 않아"라고 말씀해 주세요. 제가 로그를 보고 바로 고쳐드립니다.

### Q. 기존의 메뉴(카테고리)를 수정하거나 삭제하고 싶어요!
**A. 데이터만 바꾸면 메뉴도 알아서 바뀝니다.**
- **수정**: 해당 카테고리를 가진 모든 글의 부서명을 새로운 이름으로 바꾸면 됩니다. (예: 모든 `Marketing`을 `Brand Marketing`으로 수정)
- **삭제**: 해당 카테고리에 속한 글을 모두 지우면, 메뉴 버튼도 **자동으로 사라집니다.**
- **초기화**: 처음에 들어있던 예시 데이터(Sales, HR 등)는 여러분이 진짜 데이터를 넣을 때 제가 **싹 지우고 새로 넣어드립니다.** 그러니 신경 쓰지 않으셔도 됩니다.
